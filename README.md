# 主管會議 KPI Dashboard

既有 Excel 三頁貼上版儀表板 - 沿用原本 KPI 通用報告檔案。

## 功能特色

### 📊 CEO 總覽
- 本月營收彙總
- 平均達成率
- 紅黃綠燈統計
- 部門紅黃綠燈總表
- 本月紅燈優先清單

### 📋 KPI 明細
完整的 KPI 資料表，包含：
- 部門、月份、分類、KPI 名稱
- 本月實績、上月實績、去年同期、本月目標
- 達成率、MoM、YoY
- 燈號（紅黃綠）與主管說明

### 📈 12 個月趨勢
- 營收趨勢
- 案量趨勢
- 服務/完成率趨勢
- 紅燈數量趨勢

### ✅ 追補計畫追蹤
專注於非綠燈 KPI 的改善行動，包含：
- 原因分析
- 追補計畫
- 負責人與期限

## 使用說明

### 為主管準備的步驟

1. **打開 Excel 檔案**
   - 沿用原本提供的 KPI 通用報告檔案
   - 照原本方式填寫，不需修改格式

2. **準備三個 Sheet 的資料**
   - Sheet 1：KPI 儀表板
   - Sheet 2：月度 KPI 資料（主要來源）
   - Sheet 3：追補計畫

3. **複製資料**
   - 每個 Sheet 選取有資料的範圍
   - 建議從 A1 開始選到資料最後一欄
   - 包含第一列標題列
   - 按 Ctrl+C 複製

4. **貼入本工具**
   - 回到 KPI Dashboard
   - 將三個 Sheet 分別貼到上方三個文字框

5. **生成儀表板**
   - 按「產生儀表板」按鈕
   - 系統會自動產生執行長看的 CEO 總覽

## 計算邏輯

### 達成率 (Achievement)
- **越高越好**：實績 / 目標 × 100%
- **越低越好**：目標 / 實績 × 100%

### 燈號判定 (Status)
- 🔴 **紅燈**：達成率 < 90% 或 MoM < -5%
- 🟡 **黃燈**：達成率 < 100% 或 MoM < 0%
- 🟢 **綠燈**：其他情況

## 資料匯出

點擊「匯出目前篩選資料」按鈕可將篩選後的資料匯出為 CSV 檔案：
- 檔案名稱：`主管會議KPI儀表板輸出.csv`
- 包含所有計算欄位（達成率、MoM、YoY、燈號）
